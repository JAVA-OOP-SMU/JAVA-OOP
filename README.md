# OOP-JAVA

# 프로젝트명: **나의 즐거운 서울 생활**

## 개요

**나의 즐거운 서울 생활**은 서울시 공공데이터를 기반으로 시민의 일상생활에 필요한 정보를 제공하는 **CLI 기반 생활 도우미 서비스입니다.**

날씨, 행사, 유동 인구 데이터를 실시간으로 조회하고, OpenAI API를 활용하여 **AI가 제안하는 옷차림, 활동, 식사 메뉴**를 추천합니다.

단순한 정보 조회를 넘어 **AI + 공공데이터 융합 서비스**의 가능성을 보여주는 프로젝트입니다.

---

## 개발 목표

- **객체지향 프로그래밍(OOP) 설계 원칙**을 충실히 적용
- API 클라이언트를 인터페이스로 분리하여 **유연한 확장 구조** 구현
- 서울시 API에 더해 **기상청 전국 날씨 API**를 적용해 다형성 시연
- OpenAI API를 연동하여 **AI 기반 맞춤형 추천 서비스** 제공
